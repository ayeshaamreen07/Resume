# Single-Page Resume Website (HTML Only)

A classic, structure-focused resume built entirely with HTML. This project demonstrates the use of tables for layout and semantic HTML tags without the help of CSS.

## Features
- Uses standard HTML tags.
- Table-Based Layout.
- Purely styled via HTML attributes.

## Setup Instructions
1. Clone the repository:
   `git clone https://github.com/ayeshaamreen07/Resume.git`
2. Navigate to the folder:
   `cd Resume`
3. Open `Resume.html` in any web browser.
